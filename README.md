# 🏨 Hotel Management System

A Full-Stack Hotel Management System developed using Spring Boot, React.js, MySQL, and JWT Authentication.

## 🚀 Features

* User Authentication & Authorization
* Room Management
* Guest Management
* Booking Management
* Billing System
* Staff Management
* Dashboard Analytics
* Feedback Management

## 🛠️ Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* Hibernate
* JPA
* JWT Authentication

### Frontend

* React.js
* Axios
* HTML
* CSS
* JavaScript

### Database

* MySQL

## ⚙️ Installation

### Backend

```bash
mvn clean install
mvn spring-boot:run
```

### Frontend

```bash
cd hotel-frontend
npm install
npm start
```
